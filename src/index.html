LPA ecomms is an application to manage invoicing information: clients, stock, sales and users of the system. Here is a brief guidance of the use of the system:

Query Stock Items

To manage the Stock information you use the menu Menu > Stock Management. It takes you to a screen in which you can search for any item of the stock using either the its ID or name. After pressing the “Search” button, the system will display a list of Stock items that match with the criteria type in the field “Search”:

Create/Edit a Stock Item

On the “Search Stock” screen you have the possibility of editing an existing item (selecting it from the list) or creating a new one (button “New”). You can type the regarding information and press the button “Save”. So, the information will be recorded in the system:

 Query Sales or Invoices

To manage the Stock information you use the menu Menu > Sales and Invoicing > Invoices. It takes you to a screen in which you can search invoices typing the searching criteria in the field “Search”. The system will look for invoices whose ID, client name or date match with the typed criteria. The list of invoices shows the total amount of the sum of the all invoices on the list in the last row :

Create a New invoice

On the “Search Sales” screen you have the possibility of creating a new one (button “New”). To create a new invoice you have to search the client, next, search one by one for the item that you want to invoice, and finally pressing the button “Add Item”. At the end you press the button “Save” to record the invoice:

Query Clients

To manage the Client's information you use the menu Menu > Sales and Invoicing > Clients. It takes you to a screen in which you can search clients typing the searching criteria in the field “Search”. The system will look for clients whose ID, or client name match with the typed criteria:

Create/Edit a Client

On the “Search Clients” screen you have the possibility of editing an existing client (selecting it from the list) or creating a new one (button “New”). You can type the regarding information and press the button “Save”. So, the information will be recorded in the system:

Query Users
